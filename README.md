## CARA INSTALL DI TERMUX
```bash
> termux-setup-storage
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install bash
> pkg install ffmpeg
> pkg install libwebp
> git clone https://github.com/Botynyakamu/zifabotz-V5
> cd zifabotz-V5
> npm i
> node run.js
# Untuk Sdcard (File Sudah Di Download)
> cd /sdcard
> cp -r zifabotz-V5 $HOME
> cd zifabotz-V5
> npm i
> node run.js
```
